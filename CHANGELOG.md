# 更新日志

所有重要的项目变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-01-02

### 新增
- ✨ 初始版本发布
- 📥 一键导出B站视频列表到CSV
- ⌨️ 快捷键支持 (Ctrl+Shift+E)
- 🎨 侧边悬浮按钮UI
- 💾 导出完整视频信息（标题、BV号、链接、封面、时长、播放量、发布时间）
- 🔒 使用Data URI绕过CSP限制
- 📋 下载失败时自动复制到剪贴板
- 🚀 支持Tampermonkey和控制台两种使用方式

### 技术实现
- DOM解析提取视频信息
- CSV格式导出（UTF-8 BOM）
- 自动去重处理
- 错误处理和备用方案
